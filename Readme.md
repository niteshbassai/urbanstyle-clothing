# UrbanStyle Clothing

UrbanStyle Clothing is an online fashion store designed to showcase modern clothing, footwear, and accessories.

## Project Purpose

The purpose of this project is to create a customer-friendly e-commerce website where users can browse fashion products and learn more about the brand.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Git and GitHub
- Visual Studio Code

## Website Pages

- Homepage
- About Page
- Shop Page
- Contact Page
- Product Pages

## Setup Instructions

1. Clone the repository
2. Open the project in Visual Studio Code
3. Open index.html in a browser
